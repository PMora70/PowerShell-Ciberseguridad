# PowerShell-Ciberseguridad

Este repositorio contiene una serie de ejercicios prácticos desarrollados para aprender la automatización y auditoría de sistemas mediante PowerShell.

## Ejercicios incluidos

### Ejercicio 1: Escáner de procesos
Este script nos permite listar los procesos que se están ejecutando en el sistema, ordenándolos por uso de CPU. Es una herramienta básica pero fundamental para identificar procesos inusuales o consumo excesivo de recursos, lo cual es vital en tareas para monitoreo de seguridad.

### Ejercicio 2: Auditoría de información del sistema
Este script extrae información clave sobre el sistema operativo y el hardware del equipo. Se utiliza para documentar el estado técnico de una máquina de forma rápida, siendo una base para auditorías de inventarios de equipos.
